# Cursor Integration

Cursor is an AI-powered code editor that can significantly enhance your productivity when developing with the gx402 SDK.

## Getting Started with Cursor

### Installation

1. Download Cursor from [cursor.sh](https://cursor.sh)
2. Install the application for your operating system
3. Launch Cursor and sign in to your account

### Project Setup

1. Open your project directory in Cursor
   - `File` > `Open Folder` 
   - Select your game project directory
2. Ensure proper language support:
   - For WebGL projects: JavaScript/TypeScript extensions
   - For Unity: C# extensions
   - For Unreal: C++ extensions

Cursor adapts suggestions based on your platform:

#### WebGL/JavaScript
```javascript
// For web games, Cursor suggests browser-specific patterns
import { Gx402SDK } from '@gx402/web-sdk';

class WebGameManager {
  constructor() {
    this.isBrowser = typeof window !== 'undefined';
  }

      throw new Error('Cannot initialize in non-browser environment');
```

### Example Workflow

#### 1. Generate Initial Implementation
> "Create a player manager class that syncs position using gx402 SDK"

```javascript
class PlayerManager {
  constructor(gx402) {
    this.gx402 = gx402;
    this.playerId = null;
    this.position = { x: 0, y: 0 };
  }
  
  async initialize(playerId) {
    this.playerId = playerId;
    
    // Subscribe to other players' positions
    this.gx402.subscribe('player-position', (data) => {
      if (data.playerId !== this.playerId) {
        this.updateOtherPlayer(data);
      }
    });
  }
  
  async updatePosition(x, y) {
    this.position = { x, y };
    
    // Sync to other players
    await this.gx402.sync('player-position', {
      playerId: this.playerId,
      position: this.position,
      timestamp: Date.now()
    });
  }
  
  updateOtherPlayer(data) {
    // Update UI with other player's position
    console.log(`Player ${data.playerId} at ${data.position.x}, ${data.position.y}`);
  }
}
```
