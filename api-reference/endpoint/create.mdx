# Create Payment

This endpoint allows you to initiate a new payment within the gx402 system.

### Endpoint
```
POST /payments
```

### Request Body

```json
{
  "itemId": "string",
  "price": "decimal",
  "currency": "USDC",
  "playerWallet": "0x...",
  "developerWallet": "0x...",
  "network": "base",
  "metadata": {
    // ...
  }
}
```

### Response

```json
{
  "paymentId": "uuid",
  "status": "pending",
  "signatureRequired": true,
  "signaturePayload": {
    // ...
  }
}
```
