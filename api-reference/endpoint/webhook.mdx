# Webhook Events

This section details the webhook events that gx402 can send to your configured endpoint.

### Endpoint
```
POST {developer_webhook_url}
```

### Body

```json
{
  "event": "payment.succeeded",
  "paymentId": "uuid",
  "data": { ... },
  "signature": "HMAC-SHA256"
}
```
