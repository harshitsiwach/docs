  The gx402 SDK provides comprehensive APIs for integrating x402 functionality across multiple platforms including Unity, Unreal Engine, WebGL, Mobile, Telegram, and Farcaster.
</Note>

## Overview

The gx402 SDK API provides developers with access to powerful x402 functionality for games. This reference documents all available methods, parameters, and return values.

## Authentication

All API requests require authentication using your API key. Include your API key in the SDK initialization:

```javascript
const gx402 = new Gx402SDK({
  apiKey: 'your-api-key'
});
```

## Base URLs

- Production: `https://api.gx402.com/v1`
- Development: `https://dev-api.gx402.com/v1`

## SDK Methods

The SDK provides the following core methods:

### Core Methods
- `initialize()` - Initialize the SDK
- `authenticate()` - Authenticate users
- `processX402Data()` - Process data with x402 functionality
- `sync()` - Synchronize data across platforms
- `subscribe()` - Subscribe to real-time updates

### Platform-Specific Methods
Each platform SDK includes additional methods optimized for that platform.

## Error Handling

The SDK uses standard error handling patterns:

```javascript
try {
  const result = await gx402.processX402Data(data);
} catch (error) {
  console.error('gx402 error:', error.code, error.message);
}
```

## Rate Limits

API requests are rate-limited to ensure service availability:
- Standard requests: 100 per minute
- Data sync operations: 50 per minute
- Authentication requests: 20 per minute
