# Getting Started with All Platforms

The gx402 SDK supports multiple platforms for gaming, each with their own setup process but sharing the same core x402 functionality. This guide walks you through getting started on each platform.

## Choose Your Platform

<TabItem value="unity" label="Unity">

### Unity Setup

#### Installation

To install the gx402 SDK in your Unity project, you have two primary options:

##### Via Unity Package Manager (Git URL)

1.  Open your Unity project.
2.  Go to `Window` > `Package Manager`.
3.  Click the `+` button in the top-left corner.
4.  Select "Add package from git URL...".
5.  Enter the gx402 SDK repository URL: `https://github.com/gx402/unity-sdk.git`.
6.  Click "Add".

##### Via Unity Asset Store

1.  Open the Unity Asset Store from within Unity or your web browser.
2.  Search for "GX402 Payment SDK".
3.  Follow the instructions to download and import the package into your project.

#### Minimal Implementation Example

Here's a basic example of how to integrate the gx402 SDK into your Unity project to handle in-game purchases.

```csharp
using GX402;
using UnityEngine;

public class GameStore : MonoBehaviour
{
    private GX402Client client;

    void Start()
    {
        client = new GX402Client(
            receiverAddress: "OxYourWalletAddress",
            network: Network.Base,
            facilitatorUrl: "https://api.gx402.io"
        );
    }

    public async void BuyItem(string itemId, decimal price)
    {
        try
        {
            var result = await client.Purchase(new PurchaseRequest
            {
                ItemId = itemId,
                Price = price,
                Currency = "USDC",
                Metadata = new { userId = GetUserId() }
            });

            // Payment successful!
            GrantItemToPlayer(itemId);
            ShowNotification($"Purchased {itemId}!");
        }
        catch (GX402Exception ex)
        {
            Debug.LogError($"Payment failed: {ex.Message}");
            ShowErrorDialog(ex.UserFriendlyMessage);
        }
    }

    // Placeholder methods - implement your actual game logic here
    private string GetUserId() { return "player123"; }
    private void GrantItemToPlayer(string itemId) { Debug.Log($"Granting {itemId} to player."); }
    private void ShowNotification(string message) { Debug.Log($"Notification: {message}"); }
    private void ShowErrorDialog(string message) { Debug.LogError($"Error Dialog: {message}"); }
}
```

### Unreal Engine Setup

#### Installation

To install the gx402 SDK in your Unreal Engine project, you have two primary options:

##### Download from Unreal Marketplace

1.  Open the Unreal Engine Marketplace from within the Unreal Editor or your web browser.
2.  Search for "GX402 Payment SDK".
3.  Follow the instructions to download and add the plugin to your project.

##### Clone Source

1.  Open your terminal or command prompt.
2.  Navigate to your project's `Plugins` directory (create it if it doesn't exist).
3.  Clone the gx402 Unreal SDK repository: `git clone https://github.com/gx402/unreal-sdk.git`
4.  Copy the cloned `gx402-sdk` folder into your project's `Plugins` directory.
5.  Restart the Unreal Editor and enable the plugin in `Edit` > `Plugins`.

#### Minimal Implementation Example (Blueprint)

Here's a basic example of how to integrate the gx402 SDK into your Unreal Engine project using Blueprints to handle in-game purchases.

```blueprint
[Begin Play]
    ↓
[Create GX402 Client]
    ↓
    Receiver Address: OxYourAddress
    Network: Base
    ↓
[On Button Click: Buy Sword]
    ↓
[Call: GX402 Purchase (Async)]
    ↓
    Item ID: "legendary_sword"
    Price: 2.50
    Currency: USDC
    ↓
[On Success]  ←  [Grant Item]  ←  [Show Success UI]
[On Failure]  ←  [Show Error UI]
```

### WebGL Setup

#### Installation

Install the gx402 SDK using npm or yarn:

```bash
npm install @gx402/sdk
# or
yarn add @gx402/sdk
```

#### Minimal Implementation Example

Here's a basic example of how to integrate the gx402 SDK into your WebGL or web-based project, specifically for a Telegram Mini App, to handle in-game purchases.

```javascript
import { GX402Client } from '@gx402/sdk';

const gx402 = new GX402Client({
    receiverAddress: 'OxYourWalletAddress',
    network: 'base',
    // Auto-detects Telegram environment
    walletProvider: 'telegram',
});

// Buy boost in game
async function buyBoost() {
    try {
        const tx = await gx402.purchase({
            itemId: 'speed_boost_10min',
            price: '0.25', // $0.25
            currency: 'USDC',
        });
        console.log('Payment successful:', tx.hash);
        applyBoost('speed', 600); // 10 minutes
    } catch (error) {
        console.error('Payment failed:', error);
        showToast('Payment failed. Please try again.');
    }
}

// Placeholder functions - implement your actual game logic here
function applyBoost(type, duration) { console.log(`Applying ${type} boost for ${duration} seconds.`); }
function showToast(message) { console.log(`Toast: ${message}`); }
```

### Mobile Setup

#### Installation

Install the gx402 SDK for React Native using npm or yarn:

```bash
npm install @gx402/react-native
# or
yarn add @gx402/react-native
```

##### iOS Additional Setup

After installing the npm package, navigate to your `ios` directory and install pods:

```bash
cd ios && pod install
```
