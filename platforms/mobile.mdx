# Mobile Integration

The gx402 SDK provides native support for mobile platforms including iOS and Android, bringing x402 functionality to mobile games and applications.

## iOS Installation

### CocoaPods

Add the following to your `Podfile`:

```ruby
pod 'Gx402SDK'
```

Then run:
```bash
pod install
```

### Swift Package Manager

In Xcode, go to File > Add Package Dependencies and enter:
```
https://github.com/gx402/ios-sdk
```

## Android Installation

### Gradle

Add the following to your `build.gradle` (Module: app):

```gradle
dependencies {
    implementation 'com.gx402:sdk:1.0.0'
}
```

### Maven

```xml
<dependency>
  <groupId>com.gx402</groupId>
  <artifactId>sdk</artifactId>
  <version>1.0.0</version>
  <type>aar</type>
</dependency>
```

## React Native / Expo Installation

Install the gx402 SDK for React Native using npm or yarn:

```bash
npm install @gx402/react-native
# or
yarn add @gx402/react-native
```

### iOS Additional Setup

After installing the npm package, navigate to your `ios` directory and install pods:

```bash
cd ios && pod install
```

## Basic Setup

### React Native (JavaScript/TypeScript)

```javascript
import { GX402Client } from '@gx402/react-native';

const gx402 = new GX402Client({
    receiverAddress: 'OxYourWalletAddress',
    network: 'base',
    // Auto-detects mobile environment
});

// Buy boost in game
async function buyBoost() {
    try {
        const tx = await gx402.purchase({
            itemId: 'speed_boost_10min',
            price: '0.25', // $0.25
            currency: 'USDC',
        });
        console.log('Payment successful:', tx.hash);
        applyBoost('speed', 600); // 10 minutes
    } catch (error) {
        console.error('Payment failed:', error);
        showToast('Payment failed. Please try again.');
    }
}

// Placeholder functions - implement your actual game logic here
function applyBoost(type, duration) { console.log(`Applying ${type} boost for ${duration} seconds.`); }
function showToast(message) { console.log(`Toast: ${message}`); }
```
