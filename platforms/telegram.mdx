# Telegram Integration

The gx402 SDK provides seamless integration with Telegram mini apps, bringing x402 functionality to the Telegram ecosystem.

## Installation

### NPM Package

```bash
npm install @gx402/telegram-sdk
```

### Telegram Mini App Setup

1. Create a new Telegram bot using @BotFather
2. Set up your mini app using the Telegram Mini Apps platform
3. Install the gx402 SDK in your project

## Basic Setup

### Initialize SDK

```javascript
import { Gx402TelegramSDK } from '@gx402/telegram-sdk';
import { Telegram } from '@twa-dev/sdk';

// Initialize Telegram Web Apps first
const tg = window.Telegram.WebApp;

// Initialize gx402 SDK
const gx402 = new Gx402TelegramSDK({
  apiKey: 'your-api-key',
  telegramWebApp: tg,
  enableRealtimeSync: true
});

// Initialize the SDK
gx402.init()
  .then(() => {
    console.log('gx402 Telegram SDK initialized');
    // SDK is ready to use
  })
  .catch((error) => {
    console.error('Failed to initialize SDK:', error);
  });
```

## Core Features

### User Authentication
```javascript
// Automatically authenticate using Telegram user data
gx402.authenticateTelegramUser()
  .then((authResult) => {
    console.log('User authenticated:', authResult.userId);
  })
  .catch((error) => {
    console.error('Authentication failed:', error);
  });
```

### x402 Functionality in Telegram
```javascript
// Process game data with x402 functionality
const result = await gx402.processX402Data({
  telegramUserId: tg.initDataUnsafe.user?.id,
  gameData: yourGameData,
  options: {
    syncWithTelegram: true
  }
});
```

### Telegram-Specific Features
```javascript
// Access Telegram-specific functionality
const telegramData = gx402.getTelegramData();

// Share game results to Telegram
gx402.shareToTelegram({
  text: 'Check out my high score!',
  gameUrl: 'https://t.me/yourbot/yourapp'
});
```

## Telegram Mini App Components

### Using Telegram UI Components
```javascript
// Show a Telegram-specific alert
gx402.showTgAlert({
  title: 'Game Notification',
  message: 'New x402 event occurred!',
  button: 'OK'
});

// Use Telegram's main button
gx402.setupMainButton({
  text: 'Play Now',
  color: '#16A34A',
  onClick: () => {
    // Trigger x402 functionality
    gx402.processGameplay();
  }
});
```

## Web App Integration

### Haptic Feedback
```javascript
// Use Telegram's haptic feedback with x402 events
gx402.on('x402-event', () => {
  tg.HapticFeedback.impactOccurred('medium');
});
```

### Biometric Authentication
```javascript
// Optional biometric verification for sensitive x402 operations
const biometricResult = await gx402.requestBiometricAuth({
  prompt: 'Verify your identity to access x402 features'
});
```

## Best Practices

- Always respect Telegram's privacy guidelines
- Implement proper error handling for network failures
- Optimize for mobile performance
- Use Telegram's native UI components where possible
- Follow Telegram's design guidelines
- Test the mini app on different devices
