# WebGL Integration

The gx402 SDK provides full JavaScript support for WebGL applications, enabling x402 functionality in web-based games and experiences.

## Installation

### NPM Package

Install the gx402 Web3 SDK using npm or yarn:

```bash
npm install @gx402/sdk
# or
yarn add @gx402/sdk
```

## Minimal Implementation Example (Telegram Mini App)

Here's a basic example of how to integrate the gx402 SDK into your WebGL or web-based project, specifically for a Telegram Mini App, to handle in-game purchases.

```javascript
import { GX402Client } from '@gx402/sdk';

const gx402 = new GX402Client({
    receiverAddress: 'OxYourWalletAddress',
    network: 'base',
    // Auto-detects Telegram environment
    walletProvider: 'telegram',
});

// Buy boost in game
async function buyBoost() {
    try {
        const tx = await gx402.purchase({
            itemId: 'speed_boost_10min',
            price: '0.25', // $0.25
            currency: 'USDC',
        });
        console.log('Payment successful:', tx.hash);
        applyBoost('speed', 600); // 10 minutes
    } catch (error) {
        console.error('Payment failed:', error);
        showToast('Payment failed. Please try again.');
    }
}

// Placeholder functions - implement your actual game logic here
function applyBoost(type, duration) { console.log(`Applying ${type} boost for ${duration} seconds.`); }
function showToast(message) { console.log(`Toast: ${message}`); }
```

## Core Features

*   **Framework-agnostic**: Works with popular frameworks like React, Vue, and Svelte.
*   **Telegram Mini App optimized**: Specifically designed for seamless integration with Telegram Mini Apps.
*   **Coinbase Smart Wallet integration**: Supports Coinbase Smart Wallet for enhanced user experience.
*   **Farcaster Frame actions support**: Enables interactive experiences within Farcaster Frames.
*   **NPM package distribution**: Easily integrate into your project using npm.
