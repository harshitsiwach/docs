## Get started with gx402 in three steps

Integrate the gx402 SDK into your game and start using x402 functionality across multiple platforms.

### Step 1: Install the SDK

<AccordionGroup>
  <Accordion icon="copy" title="Choose your platform">
    Select the platform you're developing for:
    
    <Tabs>
      <TabItem value="unity" label="Unity">
        Install via Unity Package Manager:
        1. Open your Unity project
        2. Go to `Window` > `Package Manager`
        3. Click the `+` button and select "Add package from git URL..."
        4. Enter: `https://github.com/gx402/unity-sdk.git`
      </TabItem>
      <TabItem value="unreal" label="Unreal Engine">
        Install the plugin:
        1. Download from [Releases](https://github.com/gx402/unreal-sdk/releases)
        2. Place in `YourProject/Plugins/gx402-sdk`
        3. Enable in Edit > Plugins > Project > gx402 SDK
      </TabItem>
      <TabItem value="webgl" label="WebGL">
        Install via NPM:
        ```bash
        npm install @gx402/web-sdk
        ```
      </TabItem>
      <TabItem value="mobile" label="Mobile">
        For iOS (CocoaPods):
        ```ruby
        pod 'Gx402SDK'
        ```
        For Android (Gradle):
        ```gradle
        implementation 'com.gx402:sdk:1.0.0'
        ```
      </TabItem>
    </Tabs>
  </Accordion>
  <Accordion icon="key" title="Get your API key">
    1. Sign up at [https://gx402.com](https://gx402.com)
    2. Create a new project in your dashboard
    3. Copy your API key from the project settings
    4. Store it securely (never commit to source control!)
  </Accordion>
</AccordionGroup>

### Step 2: Initialize the SDK

<AccordionGroup>
  <Accordion icon="code" title="Basic initialization">
    Initialize the SDK in your game's startup code:
    
    <Tabs>
      <TabItem value="javascript" label="JavaScript/WebGL">
        ```javascript
        import { Gx402SDK } from '@gx402/web-sdk';
        
        const gx402 = new Gx402SDK({
          apiKey: 'your-api-key',
          enableRealtimeSync: true
        });
        
        await gx402.init();
        console.log('gx402 SDK initialized!');
        ```
      </TabItem>
      <TabItem value="csharp" label="C# (Unity)">
        ```csharp
        using Gx402;
        
        Gx402Manager.Initialize("your-api-key");
        
        Gx402Manager.Configure(new Gx402Config {
            EnableRealtimeSync = true
        });
        ```
      </TabItem>
      <TabItem value="cpp" label="C++ (Unreal)">
        ```cpp
        #include "Gx402SDK.h"
        
        FGx402Config Config;
        Config.ApiKey = TEXT("your-api-key");
        Config.bEnableRealtimeSync = true;
        
        UGx402Manager::Initialize(Config);
        ```
      </TabItem>
      <TabItem value="swift" label="Swift (iOS)">
        ```swift
        import Gx402SDK
        
        let config = Gx402Config(
            apiKey: "your-api-key",
            environment: .production
        )
        
        Gx402Manager.shared.initialize(with: config) { success, error in
            if success {
                print("gx402 SDK initialized successfully")
            }
        }
        ```
      </TabItem>
    </Tabs>
  </Accordion>
  <Accordion icon="user" title="Authenticate users">
    Authenticate users to enable personalized x402 functionality:
    
    ```javascript
    // JavaScript example
    await gx402.authenticate('user-id', 'auth-token');
    
    // Now you can use personalized x402 features
    const result = gx402.processX402Data(gameData);
    ```
  </Accordion>
</AccordionGroup>

### Step 3: Start using x402 features

<Accordion icon="rocket" title="Enable x402 functionality">
  Use x402 features in your game:
  
  ```javascript
  // Process game data with x402 functionality
  const x402Result = await gx402.processX402Data({
    playerAction: 'jump',
    gameState: currentGameState,
    timestamp: Date.now()
  });
  
  // Use real-time sync
  gx402.subscribe('game-state', (data) => {
    updateGameState(data);
  });
  
  console.log('x402 features enabled!');
  ```
</Accordion>
