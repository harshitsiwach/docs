# Hello World with gx402

In this tutorial, you'll create your first application using the gx402 SDK. We'll build a simple game that demonstrates basic x402 functionality.

## What You'll Build

A simple counter game where players can increment a number, with real-time synchronization across all players.

## Prerequisites

- Basic knowledge of your chosen platform (JavaScript, C#, C++, etc.)
- Node.js (for WebGL), Unity (for Unity), Unreal Engine (for Unreal), etc.
- Your gx402 API key

## Step 1: Set Up Your Project

### For WebGL/JavaScript
First, create a new directory and initialize a project:

```bash
mkdir gx402-hello-world
cd gx402-hello-world
npm init -y
npm install @gx402/web-sdk
```

Create an `index.html` file:

```html
<!DOCTYPE html>
<html>
<head>
    <title>gx402 Hello World</title>
</head>
<body>
    <h1>gx402 Counter Game</h1>
    <div id="counter">0</div>
    <button id="increment">Increment</button>
    <button id="reset">Reset</button>
    <div id="status">Initializing...</div>
    
    <script src="main.js"></script>
</body>
</html>
```

### For Unity
1. Create a new Unity project
2. Install the gx402 SDK via Package Manager
3. Create a new scene with UI elements for the counter

### For Unreal Engine
1. Create a new Unreal Engine project
2. Install the gx402 plugin
3. Create UI with UMG or create C++ classes

## Step 2: Initialize the SDK

Create a `main.js` file for the web version:

```javascript
import { Gx402SDK } from '@gx402/web-sdk';

// Initialize the SDK
const gx402 = new Gx402SDK({
  apiKey: 'YOUR_API_KEY_HERE',
  enableRealtimeSync: true,
  debug: true
});

// DOM elements
const counterElement = document.getElementById('counter');
const incrementButton = document.getElementById('increment');
const resetButton = document.getElementById('reset');
const statusElement = document.getElementById('status');

// Initialize the SDK
gx402.init()
  .then(() => {
    statusElement.textContent = 'Connected to gx402!';
    startGame();
  })
  .catch(error => {
    statusElement.textContent = `Error: ${error.message}`;
  });
```

## Step 3: Implement Basic Counter Functionality

Add the game logic to your main.js:

```javascript
// Current game state
let gameState = {
  counter: 0
};

// Start the game
async function startGame() {
  // Set up button event listeners
  incrementButton.addEventListener('click', async () => {
    // Increment counter using x402 functionality
    const result = await incrementCounter();
    console.log('Counter incremented to', result.counter);
  });
  
  resetButton.addEventListener('click', async () => {
    // Reset counter using x402 functionality
    const result = await resetCounter();
    console.log('Counter reset to', result.counter);
  });
  
  // Load initial state
  await loadGameState();
}

// Increment the counter with x402 processing
async function incrementCounter() {
  // Update local state
  gameState.counter += 1;
  
  // Process with x402 functionality
  const processed = await gx402.processX402Data({
    action: 'increment',
    currentCounter: gameState.counter
  });
  
  // Update game state and UI
  gameState.counter = processed.result || gameState.counter;
  updateUI();
  
  // Sync state across platforms
  await syncGameState();
  
  return gameState;
}

// Reset the counter with x402 processing
async function resetCounter() {
  // Update local state
  gameState.counter = 0;
  
  // Process with x402 functionality
  const processed = await gx402.processX402Data({
    action: 'reset',
    currentCounter: gameState.counter
  });
  
  // Update game state and UI
  gameState.counter = processed.result || gameState.counter;
  updateUI();
  
  // Sync state across platforms
  await syncGameState();
  
  return gameState;
}

// Update the UI with current state
function updateUI() {
  counterElement.textContent = gameState.counter;
}
```

## Step 4: Add Real-time Sync

Add synchronization functionality:

```javascript
// Subscribe to game state updates from other players
function subscribeToUpdates() {
  gx402.subscribe('game-state', (data) => {
    if (data.playerId !== 'local-player') { // Don't update for local changes
      console.log('Received remote update:', data);
      gameState = { ...data.state };
      updateUI();
    }
  });
}

// Sync current game state
async function syncGameState() {
  await gx402.sync('game-state', {
    state: gameState,
    playerId: 'local-player',
    timestamp: Date.now()
  });
}

// Load initial game state
async function loadGameState() {
  try {
    // Try to load existing state
    const queryResult = await gx402.query('game-state', {
      orderBy: { timestamp: 'desc' },
      limit: 1
    });
    
    if (queryResult.length > 0) {
      gameState = queryResult[0].state;
      console.log('Loaded game state:', gameState);
    } else {
      console.log('Starting fresh game state');
    }
    
    updateUI();
    subscribeToUpdates();
  } catch (error) {
    console.error('Error loading game state:', error);
  }
}
```

## Step 5: Add x402 Enhancement

Add some special x402 functionality to make the counter more interesting:

```javascript
// Enhanced increment with x402 features
async function incrementCounter() {
  // Update local state
  gameState.counter += 1;
  
  // Process with enhanced x402 functionality
  const x402Params = {
    action: 'increment',
    value: gameState.counter,
    metadata: {
      eventType: 'counter-increment',
      platform: 'web',
      timestamp: Date.now()
    }
  };
  
  const processed = await gx402.processX402Data({
    ...x402Params,
    x402Features: ['validation', 'enhancement', 'analytics']
  });
  
  // Apply any x402 enhancements
  gameState.counter = processed.enhancedValue || processed.result || gameState.counter;
  
  // Add special effects for certain numbers
  if (gameState.counter % 10 === 0) {
    statusElement.textContent = `Achievement! Counter reached ${gameState.counter} with x402!`;
    // Add special x402 effect
    applyX402Effect();
  }
  
  updateUI();
  await syncGameState();
  
  return gameState;
}

// Apply visual effects when x402 features are active
function applyX402Effect() {
  counterElement.style.animation = 'pulse 0.5s';
  setTimeout(() => {
    counterElement.style.animation = '';
  }, 500);
}
```

## Platform-Specific Enhancements

### Unity Version
For Unity, you'd create similar functionality using C#:

```csharp
using UnityEngine;
using UnityEngine.UI;
using Gx402;

public class Gx402HelloWorld : MonoBehaviour
{
    public Text counterText;
    public Button incrementButton;
    public Button resetButton;
    
    private int counter = 0;
    
    void Start()
    {
        Gx402Manager.Initialize("YOUR_API_KEY");
        Gx402Manager.Configure(new Gx402Config {
            EnableRealtimeSync = true
        });
        
        incrementButton.onClick.AddListener(IncrementCounter);
        resetButton.onClick.AddListener(ResetCounter);
    }
    
    async void IncrementCounter()
    {
        counter++;
        counterText.text = counter.ToString();
        
        // Process with x402
        var result = await Gx402Manager.ProcessX402Data(counter);
        counter = (int)result;
        
        // Sync across platforms
        await Gx402Manager.Sync("game-state", new { counter = counter });
    }
    
    async void ResetCounter()
    {
        counter = 0;
        counterText.text = counter.ToString();
        
        // Process with x402
        var result = await Gx402Manager.ProcessX402Data(counter);
        counter = (int)result;
        
        // Sync across platforms
        await Gx402Manager.Sync("game-state", new { counter = counter });
    }
}
```
